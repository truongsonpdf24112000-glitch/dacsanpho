# Bảo mật

## Báo cáo lỗ hổng

Nếu phát hiện lỗ hổng bảo mật, vui lòng tạo issue private hoặc liên hệ maintainer.

**Không** tạo issue công khai cho lỗi bảo mật.

## Phiên bản hỗ trợ

| Phiên bản | Hỗ trợ |
|-----------|--------|
| `master` | ✅ |

## Phạm vi

Dự án là static site, không xử lý dữ liệu người dùng. Các vấn đề bảo mật chủ yếu liên quan đến:
- Dependencies (Python packages)
- GitHub Pages configuration
